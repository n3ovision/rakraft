# 5×P – Stuptup Projektplan

**Produkt:** Premium beef jerky från äkta entrecôte  
**USP:** 40g protein/70g, 0% tillsatser, gammaldags rökning  
**Målmarknad:** Egypten (primär), Sverige (test)  
**Kanal:** D2C via influencers, e-handel

---

## Prioriterade Åtgärder

### Akut – Vecka 1-2
- [ ] Färdigställa förpackningsdesign för tryckeri
- [ ] Kontakta förpackningsleverantör
- [ ] Bekräfta produktionskapacitet

### Kortsiktigt – Vecka 3-4  
- [ ] Lansera förbättrad webbplats
- [ ] Påbörja Egypten-marknadsresearch
- [ ] Kontakta influencers

### Medellång sikt – Månad 2-3
- [ ] Produktionsprov & kvalitetskontroll
- [ ] Importlicenser Egypten
- [ ] Pilotlansering
